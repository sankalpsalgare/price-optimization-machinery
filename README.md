# Price Optimization for Agricultural Machinery Retailer

End-to-end data science project to model price elasticity and optimize pricing
for an Indian irrigation & machinery shop.

## Project Highlights
- Synthetic transaction data (2019–2025)
- COVID & drought impact modeling
- Seasonality-aware pricing
- Customer-wise elasticity
- Revenue optimization

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit (planned)
